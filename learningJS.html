Thing 1

console.log

  • 1. Make sure html and js are connected. Use script src ="xyz"
  • 2. Use console.log() to output content to the browsers console
  • 3. It's that easy. You just used Javascript.

Thing2

let

Now, use 'let' to do a simple math problem and add two numbers.

  • 1. Identify variables for the browser.

    let x, y, z

  • 2. Set the values for each variable we will add together

    x = 4

    y = 10

  • 3. Last, tell the browser to add x and y.

    z = x + y

Use console.log() to see the answer

Thing 3

Expressions

Expressions are a bunch of variables used together. x + y is an expression and so is "learn Javascript".

Try logging "learn Javascript" to the browser console.

Thing 4

var

Naming variables can be useful.

Use var to total a cart with item1 costing $3 and item2 costing $4

You can use 'var item1 =' as a template.

Thing 5

let vs.var

'let'variables cannot be redeclared.

'var' variables can be redeclared.

Thing 6

const

const variables cannot be reassigned or redeclared.

Always use const unless you know the value will change

Thing 7

Data Types

Strings

A string is several characters together enclosed within 'single' or "double" quotes

Numbers

Numbers can be written with or without decimals

Booleans

Booleans are either true or false

Arrays

Arrays are lists of stuff separated by commas and enclosed in square brackets [ ]

const phones = [iPhone, Samsung, Razr]

Objects

Objects are like real objects but code. Their attributes are written in curly brackets { }

const backpack = {strapLength = 26, pockets = 14, color = red, volume = 25}

Thing 8

JS Events

Javascript events are things that happen to HTML elements. These events can be cause by a user or by the browser. For example: a user clicks a button.
